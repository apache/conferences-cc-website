What happens at Community Over Code?

Community Over Code is the official conference of the Apache Software Foundation (ASF), bringing together the communities behind Apache open source projects. The event features project-specific technical tracks, real-world production use cases, and sessions on Apache governance, release management, security practices, and open source sustainability.

Learn More

Which Apache projects are featured?

Community Over Code includes tracks dedicated to individual Apache projects such as distributed systems, data platforms, web infrastructure, and cloud-native technologies. Sessions are led by project contributors and operators sharing implementation and operational experience.

Who should attend Community Over Code?

  • Developers building with Apache software
  • Contributors and committers to ASF projects
  • Platform engineers operating Apache technologies in production
  • Open source program offices (OSPOs)
  • Legal and compliance professionals working with Apache-licensed software
